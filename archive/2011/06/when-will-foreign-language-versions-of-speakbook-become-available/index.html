When will foreign language versions of speakbook become available?

The speakbook available now can be filled in by you in any language, but we would like to like to offer fully translated, downloadable versions in as many languages as possible. As soon as they are done we will put them up on here. If you can help with the translation process, please email us patrick@speakbook.org
